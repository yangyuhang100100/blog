## 我现在会的流程

1. 在 `posts/` 新建/修改 `.md` 文件写内容
2. 本地用 `npm run docs:dev` 预览效果
3. 用 `git add . && git commit && git push` 把更新推到 GitHub
4. 等 Vercel 自动部署，刷新线上网址看到最新内容
